## Set

### Set::combine

Hash::combineに置き換え可能？

代替コードは以下
```
$data = Set::combine($msgList, '{n}.Board.id', '{n}.Board');
↓
foreach ($msgList as $msg) {
    $data[$msg['Board']['id']] = $msg['Board'];
}
```
