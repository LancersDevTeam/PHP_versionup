## PDO

PHP5.6の場合、小数のまま更新
```
$stmt->bindParam(':param1', $float, PDO::PARAM_INT);
```
PHP7.3の場合、整数に切り捨てて更新
```
$stmt->bindParam(':param1', $float, PDO::PARAM_INT);
```

対応方法
PDO::PARAM_STR にする

## String

CakeTextに変更する

```
$state = String::uuid();
```
↓
```
$state = CakeText::uuid();
```
