
## 非推奨関数対応

### each関数

```
list($similarId, $similarString) = each($similar);
↓
$similarId = key($similar);
$similarString = $similar[$similarId];
```

### split関数

- explode関数に置き換え
- preg_split関数に置き換え（正規表現が使われている場合）

```
list($name, $domain) = split('@', $email);
↓
list($name, $domain) = explode('@', $email);
```
